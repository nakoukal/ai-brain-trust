# 🛠️ Pi Coding Agent & Úkoly

Tato poznámka slouží jako rozcestník pro práci s minimalistickým agentem **Pi** a zároveň jako přehled aktuálních úkolů.

## 📥 Instalace a zdroje
* **Oficiální web:** [pi.deaf](https://pi.deaf)
* **GitHub:** [mastra-ai/mastra](https://github.com/mastra-ai/mastra)
* **Instalace:** `npm install -g @mastra/pi`

---

## ⌨️ Rychlé příkazy pro terminál
> [!info] Použití
> Spouštějte v kořenovém adresáři vašeho projektu nebo Vaultu.

- `pi` – Spustí agenta v aktuálním adresáři.
- `pi --model claude-3-5-sonnet` – Specifikace modelu.
- `/compact` – Vyčištění historie (šetří tokeny).
- `/reset` – Restart konverzace.
